# ABOUT

奇数生成とは、美しさである_____

ここには洗練された華麗で素晴らしい奇数生成コードの数々を随時展示していきます。

名コードを発見した、または作成したとお考えの方は、こちらのGithubまたは管理人のTwitter等へお問合せ下さい。
